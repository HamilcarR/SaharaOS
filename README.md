# SaharaOS
Is an operating system's kernel for x86 platform , written in C11 , from scratch .

Features implemented : 

  - Bootloader from scratch .
  - A VGA driver
  - Interrupts
  - Keyboard inputs
  - Dynamic heap memory management
  - Added timer and sound driver
